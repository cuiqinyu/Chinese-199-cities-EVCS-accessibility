# Travel Distance Data

This folder stores travel-distance accessibility records.

Expected records:

- `Nearest1to8EVCS/100m/`: city-split CSV files at the 100 m research-unit scale.
- `Nearest1to8EVCS/Merge/`: merged or aggregated SHP products for grid, county/district, city, and province scales.

Distances should be recorded in kilometres and should distinguish ordinary EVCS and fast EVCS where applicable.
